## Scope

- Team sizes: 1, 10, 50, 100 developers
- Workloads: IT programming, development support, cloud/platform engineering
- Comparison baseline: per-user SaaS subscriptions (GitHub Copilot and other provider plans)

## Output Goal

Enable management to decide one of:

- SaaS-first (per-user subscriptions)
- Hybrid (SaaS + local server for selected workloads)
- Local-first (self-hosted inference for most workloads)
